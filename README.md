# MnemonicToCN
助记词转中文,中文助记词存储更安全,无需任何权限.

## 实现原理

[bip39英文](https://github.com/bitcoin/bips/blob/master/bip-0039/english.txt)

[bip39中文](https://github.com/bitcoin/bips/blob/master/bip-0039/chinese_simplified.txt)
